Wav Marker Cutter
=================

GitHub Pagesで公開できる、WAV音声のマーカー分割ブラウザツールです。

対応言語
--------
- 日本語
- English
- 한국어

主な機能
--------
- WAVファイル読み込み
- 波形表示
- 左クリックでシーク
- 右クリックでマーカー追加
- 「現在位置にマーカー」ボタンでマーカー追加
- Ctrl + ホイールでズーム
- ホイールのみで左右スクロール
- マーカー間を自動で切り出し
- マーカーが無い場合はファイル全体を1本として書き出し
- ファイル名リストを上から順番に分割ファイルへ適用
- 左側の台本パネルでTXT読み込み / 貼り付け確認
- 半角数字3桁から始まる台本行の黄色マーカー表示
- 台本行頭の半角数字3桁をファイル名リストへ自動追加
- 「001 キャラ名「台詞」」「001 キャラ名：台詞」形式からキャラ名を自動抽出
- 台本表示 / ファイル名リストのキャラ絞り込み
- フェードイン / フェードアウト処理
- フェード時間のスライダー設定
- フェード反映済みの区間プレビュー再生
- 番号チェックボックス選択による一部書き出し
- ZIP書き出し
- 対応ブラウザでは選択フォルダへ直接保存


台本パネルについて
------------------
左側の台本パネルでは、TXTファイル読み込み、または直接貼り付けで台本を確認できます。
半角数字3桁から始まる行は黄色く表示されます。

例：
001 春久「おはよう」
002 秋久：勉強するぞ
003 春久: 了解

上記のように、行頭に半角数字3桁がある場合は、001、002、003 がファイル名リストへ自動追加されます。
「「」「：」「:」の前にある文字列はキャラ名として認識され、台本パネルとファイル名リスト上部のプルダウンで絞り込みできます。

ファイル名リストについて
------------------------
1行に1ファイル名で入力してください。
上から順番に、分割番号が小さいファイルへ適用されます。

例：
001_serif.wav
002_serif.wav
003_serif.wav

入力が足りない場合は、001.wav、002.wav のような自動連番名が使われます。
拡張子 .wav が無い場合は自動で追加されます。

書き出し対象指定について
------------------------
「全部を書き出す」のほか、「指定した番号だけ書き出す」を選べます。
「指定した番号だけ書き出す」を選ぶと、表示中の番号がチェックボックスで縦並びに表示されます。

例：
□001
□002
□003

チェックを入れた番号だけを書き出します。
キャラ絞り込み中は、ファイル名リストと同じく絞り込み結果の番号だけがチェックボックスに表示されます。
ファイル名はプレビュー上の名前がそのまま使われます。

フェード処理について
--------------------
フェードイン、フェードアウトはチェックボックスで有効化できます。
フェード時間はスライダーで 0〜5 秒の範囲で指定できます。
短い音声では、音声の長さに合わせて自動的にフェード時間を短く調整します。
プレビュー一覧の「試聴」ボタンでも、フェードを反映した状態で確認できます。

フォルダ保存について
--------------------
ブラウザの仕様上、書き出し完了後にOSのフォルダを自動で開くことはできません。
Chrome / Edge など一部ブラウザでは「選択フォルダへ直接保存」が使えます。
非対応ブラウザではZIPとして保存してください。

公開方法
--------
以下のファイルをGitHub Pagesのリポジトリ直下へ配置してください。

- index.html
- style.css
- app.js
- README.txt

外部ライブラリは使用していません。
